Projects

Stationkeeping Strategies for Earth–Moon Halo Orbits

1. Developed MATLAB and Python algorithm to compute periodic L1/L2 halo orbits in the Earth-Moon Circular Restricted 3-Body Problem using STM-based differential correction.
2. Implemented orbit propagation using adaptive Runge-Kutta and stiff ODE solvers to assess orbit stability.
3. Designed time-invariant and time-varying LQR in Simulink for improving stationkeeping performance.
4. Developed a learning-augmented framework for data-driven Stationkeeping using Ridge Regression.

X-ray spectroscopy of bright Active Galactic Nuclei observed by NASA’s Swift X-ray telescope

1. Reprocessed NASA’s Swift X-ray data of three bright active galactic nuclei (AGN) sources using HEASoft.
2. Extracted photon indices, flux values and 90% confidence range across soft and hard X-ray bands (0.3-10 keV).
3. Implemented Python scripts to quantify X-ray flux variability of the AGN sources over time.

Design of a delta-wing aircraft with Leading-Edge Root Extension

1. Developed a slender delta-wing aircraft model design with replaceable delta-nose and LERX configurations for subsonic testing of flow control.
2. Incorporated an adapter housing for the internal force balance and elevator blocks with provision for elevon integration.

Dynamics of Accretion: Viscid and Inviscid Flows

1. Developed Python code to simulate 2D incompressible viscous flow in accretion disks using phiflow package.
2. Simulated 2D cavity flow in MATLAB using finite-difference method (FDM) and pressure projection.
3. Computed spherical accretion solutions to identify critical points using phase-space and eigenvalue analysis.
